No.1

Workshops' website - CHIEU

This is a small project my friends and I've worked on together recently. The idea was designing a website which provides all the information about DIY workshops in Vietnam as well as the shortest way to book a slot to participate in these workshops. This could solve the inconvenience such as time consumed on texting the operators, vague review or any unclear questions about the workshop's space.

We started the project from scratch, did the survey, site map, empathy map; created logo and branding stages; made wireframes (low-fi and visual) with responsive for 3 devices and the UI interface with simple prototypes as the last step.
